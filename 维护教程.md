### 1. 创建文章

```bash
hexo new post "文章标题"
```

这会在 `source/_posts/` 下生成一个 `文章标题.md` 文件（如果开了 `post_asset_folder`，还会同时生成同名文件夹放图片）。

### 2. 写内容

编辑生成的 `.md` 文件，开头的 front-matter 可以改：

```markdown
---
title: 文章标题
date: 2026-06-20 12:00:00
tags:
  - 标签1
  - 标签2
categories:
  - 分类
---

正文内容，Markdown 格式。
```

图片放进同名文件夹里，文中用 `![描述](图片名.png)` 引用。

### 3. 推送上线

```bash
git add .
git commit -m "post: 文章标题"
git push
```

GitHub Actions 自动构建部署，1-2 分钟后文章就上线了。

---

 **本地预览** （推送前可以先看看效果）：

```bash
hexo server
```

浏览器打开 `localhost:4000`。

**总结就是：`hexo new` → 写 Markdown → `git push`，搞定。**
